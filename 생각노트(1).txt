//// 이미지 주소를 입력하는 방식

1. 타이틀 ("AI 도슨트: 이미지를 설명해드려요!)
2. 간단한 설명 ("여기에 이미지 주소를 입력하세요")
3. 이미지 url 입력 칸
4. 입력버튼
5. 해설이 나오는 곳 >  초록색으로 나옴

버튼을 클릭하면, 사용자 이벤트
주소를 gpt한테 줘서 이미지 입력
gpt가 이미지에 대한 해설을 입력해줌

실행 과정
1. 라이브러리 설치
2. api키 설정하기
3. client 객체 생성해서 요청하기
4. 설명을 듣는 함수를 설정
- gpt한테 요청하고 응답을 저장하는 변수 response
- model을 gpt로 지정
- role이 user인건 메시지의 역할이 사용자이고 사용자가 요청하면 gpt가 대답
- 첫번째 타입은 텍스트 유형이고 텍스트로 자세한 설명을 요청하는 것
- 두번째 타입은 image_url 유형이고 image_url 변수에 담긴 이미지 주소를 ai가 분석
